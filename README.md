# Problote v. 2.3.4
Problote est une alternative à Pronote, open source, qui ne plante pas toutes les 5 minutes
___________________________________________________________________________________________________________________________________________________

Problote est un logiciel qui offre une alternative à Pronote. Il s'agit d'un outil open source léger, doté d'un lien serveur crypté plus sécurisé.
L'équipe de Problote annonce la sortie de ce logiciel pour Windows et Linux dès 2027 !
De plus, toutes données seron acheminées via un serveur sécurisé et localisé en France.
